# 🔐 Random Password Generator

**Автор:** Левин Ярослав Антонович

## Описание программы

Графическое приложение для генерации надежных случайных паролей. Позволяет настраивать длину пароля и выбирать используемые символы (цифры, буквы верхнего/нижнего регистра, спецсимволы). История последних сгенерированных паролей сохраняется локально для удобства использования.
*Технологии:* Python 3.x и стандартная библиотека Tkinter.
> **Безопасность:** История паролей хранится в файле `history.json`, который добавлен в `.gitignore`. Не публикуйте свои пароли!

## Инструкция по запуску

**Требования:**
Установленный интерпретатор Python версии 3.6 или новее.
Библиотека Tkinter входит в стандартную поставку Python.

**Шаги:**
1. Убедитесь, что Python установлен.
2. Скачайте или склонируйте этот репозиторий.
3. Запустите скрипт из корня проекта:
   ```bash
   python main.py   # Для Windows / Linux / macOS (в виртуальном окружении)
   ```
4. Настройте параметры генерации.
5. Нажмите кнопку «Сгенерировать пароль».
6. Используйте ползунок для изменения длины. Минимальная длина — 4 символа.
7. Для очистки истории удалите файл `history.json`.
